# Epics folder (BMAD layout hint)

The [BMAD getting-started tutorial](https://github.com/bmad-code-org/BMAD-METHOD/blob/main/docs/tutorials/getting-started.md) shows epic files under `planning-artifacts/epics/`.

For **photo-tool**, the living epic and story breakdown is kept in:

- [`../epics.md`](../epics.md)

BMAD skills that search `*epic*.md` will find `epics.md`. Use this folder for future sharded epics (`index.md` + segments) if the document grows large.
