Photo Tool — design direction explorations

Non-production showcase only. The real app UI is Fyne (Go); this HTML file exists solely to compare palette/density ideas in a browser before you map choices into a fyne.Theme and tokens. It is not a UI stack decision for the desktop client. Six static wireframes for bulk review (filters + thumbnail grid). Open in a browser, pick a direction (or combine traits), then record the choice in ux-design-specification.md (Design Direction Decision).
Locked for product: Direction 1 Dark DAM default + full light/white Fyne theme (documented in the spec).
